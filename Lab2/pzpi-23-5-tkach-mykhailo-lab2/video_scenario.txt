СЦЕНАРІЙ ВІДЕО-ДЕМОНСТРАЦІЇ (Лабораторна робота №2)

1. Вступ (0:00 - 0:30)
   - "Вітаю! Я студент групи ПЗПІ-23-5 Ткач Михайло. Представляю лабораторну роботу №2 з дисципліни 'Аналіз та рефакторинг коду'."
   - "Тема роботи — розробка серверної частини системи EcoRecycle. Мною було обрано мову C++ для забезпечення максимальної швидкодії та енергоефективності."

2. Огляд структури проєкту (0:30 - 1:30)
   - (Показати дерево тек у редакторі коду)
   - "Ось структура проєкту. У папці `include` знаходяться заголовні файли: `models.h` з описом структур даних, `database.h` для роботи з SQLite, та `controllers.h`."
   - "У папці `src` — реалізація логіки. Використовуємо бібліотеку Crow для обробки HTTP запитів."
   - (Показати `CMakeLists.txt`)
   - "Для збірки використовується CMake, який підключає необхідні бібліотеки."

3. Збірка та запуск (1:30 - 3:00)
   - (Відкрити термінал)
   - "Зараз я скомпілюю проєкт. Створюємо папку build: `mkdir build && cd build`."
   - "Запускаємо CMake: `cmake ..`."
   - "Запускаємо компіляцію: `make`."
   - "Все пройшло успішно. Запускаємо сервер: `./server`."
   - "Бачимо повідомлення, що сервер запущено на порті 18080."

4. Тестування API (3:00 - 6:00)
   - (Використати Postman або curl)
   - "Перевіримо роботу API."
   - **Крок 1. Реєстрація.** "Відправляємо POST запит на `/api/auth/register` з email та паролем. Отримуємо статус 201 Created."
   - **Крок 2. Пункти прийому.** "Відправляємо GET запит на `/api/points`. Отримуємо JSON список пунктів, які були автоматично додані при ініціалізації БД."
   - **Крок 3. Здача сміття.** "Відправляємо POST запит на `/api/recycle`. Вказуємо ID користувача, тип сміття та вагу. Сервер розраховує бонуси і повертає відповідь."

5. Перевірка Бази Даних (6:00 - 7:00)
   - (Відкрити файл ecorecycle.db через SQLite Viewer або термінал)
   - "Тепер переглянемо файл бази даних. Бачимо таблицю `users` з нашим новим користувачем."
   - "І таблицю `transactions`, де з'явився запис про нашу операцію."

6. Висновки (7:00 - 7:30)
   - "Сервер працює стабільно, виконує всі поставлені задачі. Використання C++ дозволило створити швидкий та ефективний бекенд для системи EcoRecycle. Дякую за увагу!"
